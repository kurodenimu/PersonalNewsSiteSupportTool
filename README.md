# PersonalNewsSiteSupportTool
個人ニュースサイトの更新を支援するツールです。

>個人ニュースサイト（こじんニュースサイト）とは個人が運営するWebページで、主にWeb上にソースのあるニュースをそのURLにコメントを添えて読者に紹介するサイトである。

（https://ja.wikipedia.org/wiki/個人ニュースサイト より引用）
